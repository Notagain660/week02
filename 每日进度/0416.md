- 挂载 (Mount)：
    - **定义**：将 Vue 应用或组件渲染到真实的 DOM 节点上的过程。
    - **核心方法**：`createApp(App).mount('#app')` 将根组件挂载到 `index.html` 中 `<div id="app">` 所在的 DOM 元素上。
    - **内部过程**：Vue 会创建虚拟 DOM，然后转换成真实 DOM 替换掉挂载点。
    - 生命周期钩子（composition API）
        - `onMounted`：组件挂载后执行，通常用于**首次加载数据**（如获取帖子列表）。
        - `onBeforeMount`、`onUpdated`、`onUnmounted` 等用得较少。
        - 示例：在 `PostList.vue` 中 `onMounted(() => fetchPosts())`。
    - 监听（watch）
        - `watch`：监听响应式数据的变化，执行副作用。
        - 例如：监听 `route.params.postId` 变化，重新加载帖子详情。
        - `watchEffect`：自动收集依赖，立即执行。

- 连接后端的具体流程（一次请求的生命周期）
    1. 用户在登录页输入账号密码，点击登录。
    2. `handleLogin` 调用 `userStore.login` → `loginApi` → `request.post('/login', data)`。
    3. axios 实例发出请求，由于 `baseURL: '/api'`，实际 URL 是 `http://localhost:5173/api/login`。
    4. Vite 代理检测到 `/api`，将请求转发到 `http://localhost:8080/api/login`。
    5. 后端处理，返回 JSON（包含 token 和 user）。
    6. axios 响应拦截器检查 `code`，如果是 200 则返回数据。
    7. `userStore.login` 将 token 存入 state 和 localStorage，调用 `fetchUserInfo`（可选），然后返回成功。
    8. 页面跳转到首页。

- **前端 API 封装**做的事情类似：
    - `request.js` 相当于前端的“全局拦截器”，统一处理请求头（加 token）、统一处理响应错误（如 401 跳登录）。
    - 各个 `api/*.js` 模块相当于后端的 Service 层，把具体的接口调用封装成方法，供页面调用。
